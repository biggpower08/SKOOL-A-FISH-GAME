# Finished Game Notes

Date: 2026-07-09

SKOOL-A is organized as a small playable browser game first, with testing tools kept out of the public menu.

- Player Mode is the intended public game.
- Dev Mode lives in Settings for testing.
- Dev Mode keeps the level scroller, recruit test, artifact toggles, and free dev purchases.
- Player Mode hides dev controls and keeps the artifact panel as a read-only collection/manual view.
- Grog is the center-school disruptor.
- Bill is the isolated-fish hunter.
- Norman remains the baseline closest-fish hunter.
- Steezy remains the faster closest-fish hunter.

Future work after this version should stay focused on polish, tuning, content, and stability. Avoid adding new currencies, boss systems, large progression systems, or engine rewrites until this version is stable in normal player runs.
