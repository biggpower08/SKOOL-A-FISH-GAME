# Artifact Design Board

This is a tuning board for SKOOL-A-FISH-GAME artifacts. It preserves the current
50 artifact definitions from code and gives Caleb a place to revise names,
effects, upgrade behavior, icon ideas, and implementation status before the
artifact system becomes bigger.

Do not treat this as an implemented economy. `src/systems/artifacts.ts` remains
the runtime data source until a future pass intentionally changes that.

## Tone Direction

- satirical
- fish-game charm
- simple and readable
- a little hippie/new-wave
- funny without becoming cluttered
- strategy-supporting, not pure random jokes

## Categories To Keep Covered

- fish speed
- fish schooling
- fish bump/separation
- fish evasion
- fish health
- shark hunger drain
- shark speed reduction
- Shell rewards
- kelp/recovery
- recruitment
- artifact economy
- risky artifacts
- tank fish support
- fast fish support
- common fish support
- anti-shark panic tools

## Reusable Artifact Icon Prompt Template

```text
Create one original transparent-background artifact icon for SKOOL-A-FISH-GAME.

Artifact: [ARTIFACT NAME].
Function/theme: [WHAT IT DOES].
Personality: [SHORT VIBE].
Style: thick cartoon outline, simple readable shape, colorful satirical fish-game charm, compact icon readability, easy-to-cut outer edge.
Background: transparent background required with true alpha transparency.
Important transparency rule: no checkerboard pattern, no fake transparency preview, no gray/white grid background.
Avoid: text, labels, UI, water background, contact sheet, multiple objects unless required, realistic detail, fuzzy edge, cropped object, stray pixels.
Output format: one transparent PNG-style artifact icon, centered with full object visible.
```
